# 🗡️ ローグモンスターコレクション (Rogue Monster Collection)

ブラウザで遊べる、シンプルかつ奥深いローグライクRPGです。
ランダムに生成されるダンジョンを探索し、モンスターを捕獲して仲間に加え、最深部を目指しましょう！

![Game Screenshot](./screenshot.png)
*(スクリーンショットがあればここに配置)*

## ✨ 特徴 (Features)

*   **ランダム生成ダンジョン**: プレイするたびにマップの形状、敵の配置、宝箱の中身が変化します。
*   **モンスター収集**: 遭遇したモンスターを弱らせて「捕獲」し、自分のパーティー（最大3体）に加えることができます。
*   **ターン制バトル**: シンプルながら戦略的なターン制コマンドバトル。敵の属性や残りHPを考慮して戦いましょう。
*   **育成要素**: 戦闘で経験値を得てモンスターがレベルアップ。ステータスが上昇します。
*   **階層システム**: 各フロアのボスを倒すと階段が出現。深く潜るほど敵は強く、報酬も良くなります。
*   **アイテムシステム**: 宝箱や敵からのドロップでアイテムを入手。回復や蘇生を駆使して生存を目指します。

## 🎮 遊び方 (How to Play)

### 操作方法
*   **移動**: 矢印キー (`↑`, `↓`, `←`, `→`) または `W`, `A`, `S`, `D`
*   **決定/攻撃**: クリック または UIボタン操作

### ルール
1.  **探索**: 視界は限られています。暗闇を照らしながらマップを探索しましょう。
2.  **戦闘**: モンスターに接触すると戦闘開始。
    *   **攻撃**: 通常攻撃やスキルでダメージを与えます。
    *   **捕獲**: 敵のHPが減るほど捕獲成功率が上がります。
    *   **逃げる**: 危険な時は逃げることも重要です。
3.  **進行**: フロアのどこかにいる「ボス」を倒すと、次の階層への階段（梯子）が出現します。
4.  **ゲームオーバー**: パーティー全員のHPが0になるとゲームオーバーです。

## 🏆 スコアリングルール (Scoring Rules)

ゲーム終了時に以下の基準でスコアが計算されます：

### ⚔️ 戦闘スコア
モンスターを撃破した際に獲得できるスコアです。
- **計算式**: `100 × 敵のレベル × レアリティ倍率`
- **レアリティ倍率**:
  - Common（コモン）: 1.0倍
  - Uncommon（アンコモン）: 1.5倍
  - Rare（レア）: 3.0倍
- **例**: レベル3のUncommonを撃破 → 100 × 3 × 1.5 = **450点**

### 🎒 捕獲スコア
モンスターを捕獲した際に獲得できるスコアです（撃破よりも高得点）。
- **計算式**: `300 × 敵のレベル × レアリティ倍率 + 初捕獲ボーナス500点`
- **例**: レベル2のCommonを捕獲 → 300 × 2 × 1.0 + 500 = **1100点**（初捕獲時）
- **例**: 同じタイプを2回目以降に捕獲 → 300 × 2 × 1.0 = **600点**

### 💎 探索スコア
宝箱を開けた際に獲得できるスコアです。
- **計算式**: `宝箱1個につき50点`

### ✨ ボーナス
特別な行動に対して加算されるボーナススコアです。
- **初捕獲ボーナス**: 初めて捕獲したモンスタータイプごとに**500点**
  - 例: スライムを初捕獲 → 500点、ゴブリンを初捕獲 → さらに500点
- **最大ダメージボーナス**: ゲーム中の1回の攻撃で与えた最大ダメージ値をそのまま加算
  - 例: 最大ダメージ15 → **15点**

### 🏢 フロア到達ボーナス（非表示）
より深いフロアに到達することで合計スコアに加算されます。
- **計算式**: `フロア数 × 200`
- **注意**: 内訳には表示されませんが、合計スコアには含まれます

## 📦 インストールと実行 (Installation)

本ゲームは外部ファイル（JSON）を読み込む構成になる予定のため、**ローカルサーバー**上での実行を推奨します。

### 推奨: VS Code "Live Server" 拡張機能を使用する場合

1.  VS Codeの拡張機能マーケットプレイスで **"Live Server"** (Ritwick Dey作) を検索し、インストールします。
2.  VS Codeでこのプロジェクトのフォルダを開きます。
3.  右下のステータスバーにある **"Go Live"** ボタンをクリックするか、`index.html` を右クリックして **"Open with Live Server"** を選択します。
4.  自動的にブラウザが立ち上がり、ゲームが開始されます。

### その他の方法

Pythonがインストールされている場合は、以下のコマンドでも簡易サーバーを起動できます。
```bash
# プロジェクトフォルダ内で実行
python -m http.server
# ブラウザで http://localhost:8000 にアクセス
```
