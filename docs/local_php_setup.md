# ローカルPHP環境構築ガイド（コマンドライン版）

ランキングシステムをローカル環境（ご自身のPC）で動作させるための手順です。
VS Codeの拡張機能を使わず、コマンドラインで直接起動する方法です。

## 1. PHPのダウンロードと配置

1. [PHP for Windows ダウンロードページ](https://windows.php.net/download#php-8.3) にアクセスします。
2. **VS16 x64 Non Thread Safe** (または Thread Safe) の **Zip** をダウンロードします。
3. ダウンロードしたZipファイルを解凍します。
4. 解凍したフォルダを、分かりやすい場所に移動します。
   - 推奨: `C:\php`

## 2. サーバーの起動

1. VS Codeでプロジェクト（`ローグ`フォルダ）を開きます。
2. ターミナルを開きます（メニューの「ターミナル」→「新しいターミナル」、または `Ctrl + @`）。
3. 以下のコマンドを入力して **Enter** キーを押します。

   ```powershell
   C:\php\php.exe -S localhost:3000
   ```

   ※ PHPを `C:\php` 以外に置いた場合は、パスを書き換えてください。

4. ターミナルに以下のような表示が出れば起動成功です。
   ```
   [Mon Dec  2 07:22:44 2025] PHP 8.3.0 Development Server (http://localhost:3000) started
   ```

## 3. ゲームのプレイ

ブラウザで以下のURLを開いてください。
[http://localhost:3000](http://localhost:3000)

## 4. サーバーの停止

サーバーを終了するには、ターミナルで **`Ctrl + C`** を押します。

---

**トラブルシューティング:**

- **「パスが見つかりません」などのエラー**:
  - `C:\php\php.exe` の部分が、実際にPHPを置いた場所と合っているか確認してください。

- **データが保存されない場合**:
  - `data` フォルダの書き込み権限を確認してください。
